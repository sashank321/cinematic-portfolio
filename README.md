## ⚡ Overview

I built this site to combine clean editorial design with responsive, high-framerate interactions. Rather than relying on heavy template frameworks or bloated 3D models, everything is crafted using HTML5 Canvas, Lenis inertial scrolling, and GSAP timeline choreography.

### Key Features
- **Limelight Navigation**: Spotlight-tracking nav bar with scroll minimization, subtle glassmorphism, and session-aware loader skipping.
- **Interactive Particle Physics**: Canvas-rendered interactive hero text with real-time cursor repulsion dynamics.
- **Radial Case Study Carousel**: Scroll-pinned 3D timeline wheel showcasing selected projects.
- **Featured Projects**: Detailed case studies covering **Cortex AI** (Multi-Agent Orchestration), **Data Science & Analytics**, and **Java Systems Architecture**.
- **Ambient Starfield**: Dynamic, fixed canvas star background with gentle pulse animations.

---

## 🛠️ Stack & Technologies

- **Framework**: Next.js 16 (App Router + Turbopack)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4 + custom CSS design tokens
- **Animations**: GSAP 3 (ScrollTrigger), Lenis Smooth Scroll
- **Performance**: HTML5 Canvas 2D API for particle rendering

## 🚀 Running Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sashank321/Jsashank-portfolio.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Launch the development server**:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
